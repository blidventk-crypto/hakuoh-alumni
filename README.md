# 白鷗中等部同窓会サイト

このフォルダは GitHub 掲載用の完成ファイルです。

## ファイル構成

- `index.html`  
  サイト本体です。HTML / CSS / JavaScript を 1 ファイルにまとめています。
- `README.md`  
  この説明ファイルです。

## GitHub Pages で公開する場合

1. GitHub に新しいリポジトリを作成
2. このフォルダ内の `index.html` と `README.md` をアップロード
3. リポジトリの `Settings` → `Pages` を開く
4. `Build and deployment` の Source を `Deploy from a branch` に設定
5. Branch を `main`、フォルダを `/root` にして保存

数分後に GitHub Pages の URL が発行されます。

## 注意

プロフィール写真は Google Drive の公開画像 URL を参照しています。表示されない場合は、Drive 側の共有設定を「リンクを知っている全員が閲覧可」にしてください。
